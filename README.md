# 吕布校场排位工具

三国·冰河时代游戏中打吕布活动时的玩家位置排位工具。

## 功能特点

1. **Excel 文件解析**：支持上传包含玩家四维和战力数据的 Excel 表格
2. **智能排序**：根据"四维和"或"输出"列降序排列玩家数据
3. **可视化布局**：直观的 45° 平面坐标系网格展示，支持水平显示内容
4. **自定义配置**：
   - 吕布校场坐标位置（X、Y 坐标）
   - 排列环数（1环、2环、3环、4环）
5. **详细信息**：显示排名、玩家姓名、坐标、环数和位置类型
6. **导出功能**：支持将结果导出为 Excel 文件，按排名升序排列
7. **响应式设计**：适配不同屏幕尺寸
8. **忽略玩家**：支持设置玩家忽略状态，不计入排位
9. **距离级别**：根据距离吕布校场的距离设置不同颜色深浅，提高视觉识别效果
10. **统计信息**：显示总玩家数、总位置数、已分配位置和空位置数量

## 使用方法

### 1. 准备数据文件

准备包含以下列的 Excel 文件（xlsx 或 xls 格式）：
- **游戏ID**：玩家姓名
- **四维和**：四维属性总和（步维+弓维+骑维+智维）
- **步维(坦度)**：步维属性值
- **弓维(输出)**：弓维属性值

### 2. 上传文件

点击"上传 Excel 文件"按钮，选择你的数据文件。

### 3. 配置参数

- **吕布校场 X 坐标**：设置吕布校场左下角的 X 坐标（默认：5）
- **吕布校场 Y 坐标**：设置吕布校场左下角的 Y 坐标（默认：5）
- **排列环数**：选择要排列的环数（1-4环，默认：1环）
- **排位方式**：选择排位依据（四维和或输出）

### 4. 生成排位

配置完成后，系统会自动生成排位结果。

### 5. 查看结果

- **网格视图**：直观显示吕布校场和玩家位置，每个单元格水平显示内容
- **表格视图**：详细列出每个玩家的排名、姓名、四维和、坦度、输出、坐标、环数和位置类型
- **统计信息**：显示总玩家数、总位置数、已分配位置和空位置数量

### 6. 导出结果

点击"导出结果"按钮，将结果导出为 Excel 文件，按排名升序排列。

### 7. 忽略玩家

在表格视图中，勾选玩家姓名旁的"忽略"复选框，该玩家将不计入排位。

## 位置说明

### 坐标系
- X 轴：从左下到右上
- Y 轴：从右下到左上
- 坐标值从 (0, 0) 开始，不显示负数坐标的单元格

### 位置等级
- **最优位置**：吕布校场一环的四边位置（共6个）
- **次优位置**：吕布校场一环的四角位置（共4个）
- **二环及以上**：位置好坏依次类推

### 吕布校场
- 占据 2x2 个单元格
- 以其中下方最小单元格作为坐标原点
- 吕布校场单元格保持居中布局且不显示 hover 效果

## 技术实现

- **纯前端**：无需后端服务器
- **Excel 解析**：使用 SheetJS (xlsx) 库解析 Excel 文件
- **可视化**：HTML5 Canvas 网格布局
- **样式**：使用 CSS3 实现渐变、阴影等视觉效果
- **响应式**：使用 CSS Grid 和 Flexbox 布局

## 部署说明

### 本地运行

1. 安装依赖（无额外依赖）
2. 运行开发服务器：
   ```bash
   npm run dev
   ```
3. 在浏览器中访问 http://localhost:8000

### 生产部署

1. 构建项目：
   ```bash
   npm run build
   ```
2. 部署 `dist/` 目录到任何静态文件服务器
3. 或直接部署到 GitHub Pages

## 浏览器支持

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 注意事项

1. 确保 Excel 文件格式正确，包含所需列
2. 坐标系统从 (0,0) 开始
3. 建议使用最新版本的浏览器以获得最佳体验
4. 导出功能需要浏览器支持 FileSaver API
